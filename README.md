# Skill Forge 🛡️⚒️

[cite_start]O **Skill Forge** é uma plataforma de aprendizado online desenvolvida para centralizar cursos de alta qualidade em trilhas de aprendizagem estruturadas[cite: 4, 15]. [cite_start]O sistema integra gamificação através de um ranking por competência, incentivando o progresso contínuo dos estudantes[cite: 4, 16].

🔗 **Acesse o site:** [https://renzolv135.github.io/Skill_Forge/](https://renzolv135.github.io/Skill_Forge/)

## 🚀 Sobre o Projeto

[cite_start]O projeto nasceu para resolver a fragmentação de cursos na internet, que muitas vezes deixa os estudantes perdidos ou desmotivados[cite: 5, 6]. [cite_start]A plataforma foca em organizar o caminho de aprendizado e facilitar a exibição de competências para o mercado de trabalho[cite: 12, 17].

### Principais Funcionalidades 🛠️

- [cite_start]**Trilhas de Aprendizagem:** Organização lógica de cursos para guiar o aprendizado do aluno[cite: 24, 42].
- [cite_start]**Gamificação e Ranking:** Sistema de pontuação baseado na conclusão de cursos e desafios, com rankings segmentados[cite: 19, 27].
- [cite_start]**Emissão de Certificados:** Geração automática de certificados ao finalizar cursos[cite: 28, 42].
- [cite_start]**Validação Externa:** Funcionalidade para validar certificados de instituições parceiras e converter em pontos no perfil[cite: 21, 32].
- [cite_start]**Área do Recrutador:** Espaço para empresas buscarem talentos com base em desempenho e certificados[cite: 30, 42].
- [cite_start]**Interface Responsiva:** Otimizada para celulares, tablets e computadores[cite: 33, 37].

## 🧪 Tecnologias Utilizadas

[cite_start]O sistema utiliza uma arquitetura dividida em Frontend, Backend e Banco de Dados[cite: 40]:

- [cite_start]**Frontend:** HTML5, CSS3, JavaScript, Bootstrap e Font Awesome[cite: 36, 44].
- [cite_start]**Backend:** Java com Spring Boot[cite: 40].
- [cite_start]**Banco de Dados:** MySQL para armazenamento de usuários, cursos e rankings[cite: 39, 46].
- [cite_start]**Ferramentas:** Git/GitHub para versionamento e Trello (metodologia SCRUM) para gestão[cite: 1, 40].

## 📁 Estrutura do Projeto

O código está organizado da seguinte forma:

- [cite_start]`/css`: Arquivos de estilização e responsividade[cite: 46].
- [cite_start]`/js`: Lógica de interatividade e comunicação[cite: 46].
    - [cite_start]`login.js`: Gerencia o acesso diferenciado entre alunos e professores[cite: 46].
    - [cite_start]`main.js`: Administração de cursos e funcionalidades do painel[cite: 46].
