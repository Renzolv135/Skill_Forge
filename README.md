# 📌 Sobre o Projeto

O **Skill Forge** foi criado para resolver um problema muito comum no aprendizado online: a desorganização dos cursos espalhados em diferentes plataformas.

A proposta do sistema é centralizar cursos de qualidade em um único ambiente, oferecendo:

- 📚 Trilhas de aprendizagem organizadas
- 🏆 Sistema de ranking e pontuação
- 📄 Certificados automáticos
- 👤 Perfil profissional para recrutadores
- 🎯 Gamificação para manter os alunos motivados

4. Ganha pontos automaticamente
